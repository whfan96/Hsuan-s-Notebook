Linux common commands

紀錄 Linux 常用指令

基本

Command Name Documentation
date Date 查看日期
cal Calendar 列出日曆
grep Globally search for a regular expression and print matching lines 搜尋某些特定字元

網路功能

Command Name Documentation
ifconfig Interface configuration 查詢目前系統的網路卡狀況
ping Ping 查看對方網路是否有動作
hostname Hostname 查看主機名稱
nslookup From name server lookup 查詢或反查詢 DNS

系統管理員

Command Name Documentation
top Table of processes 列出目前系統的服務項目的動態資料
free Free 查看記憶體
passwd Password 改變密碼
clear Clear 清除螢幕畫面

檔案與目錄管理

Command Name Documentation
ls List 列出目錄內之檔案與子目錄
cd Change directory 改變目前目錄
pwd Print work directory 列出目前所在路徑
cat Concatenate 列出檔案內容
more More 讀取更多內容(檔案太大時)
mkdir Make directory 建立目錄
rmdir Remove directory 刪除目錄
rm Remove 刪除檔案
cp Copy 拷貝檔案
mv Move 移動檔案或目錄
find Find 列出尋找的檔案

登入與登出（開機與關機）

Command Name Documentation
telnet Teletype network 遠端登入
login Login 登入系統
logout Logout 離開作業系統
exit Exit 離開 Linux 系統
shutdown Shutdown 關機
reboot Reboot 重新開機
